# YouYanMusic
A lightweight third-party NetEase Cloud Music Android client built with Jetpack Compose and AndroidX Media3.

More features are under development. Currently, only basic account login, search, and playback functionalities are supported.

Special thanks to [Moriafly](https://github.com/Moriafly) for inspiration from [SaltUI](https://github.com/Moriafly/SaltUI) and their work [Salt Player](https://moriafly.com/program/salt-player.html) üíï
